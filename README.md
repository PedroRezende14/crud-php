# CRUD-PHP

## Pré-requisitos

- Git
- Docker
- Docker Compose
- Visual Studio Code

## Inicializar o Projeto

1. Clone este repositório:
```sh
git clone <https://github.com/PedroRezende14/crud-php.git>
```
2. Execute o seguinte comando no terminal:
```sh
docker-compose up --build -d
```
 3. Abra o navegador e digite:
```sh
http://localhost:8080/
```
